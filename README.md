# FSD
## Learning Outcomes

1. **SPA Architecture**: Understand the core concepts of Single Page Applications (SPA) and how they differ from traditional multi-page applications.
2. **Component-Based Development**: Master the creation of reusable, modular UI components using React.js to build scalable user interfaces.
3. **Client-Side Routing**: Implement seamless navigation between different views (Home, About, Contact) without page reloads.
4. **Modern Build Tools**: Gain proficiency in using modern frontend build tools like Vite for efficient development, hot module replacement, and bundling.
5. **Project Structure & Organization**: Learn best practices for structuring a full-stack project, organizing assets, and managing dependencies.
